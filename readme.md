# Data Structures and Algorithms (DSA) Documentation

Welcome to the DSA documentation! This repository contains various notes, explanations, and examples related to Data Structures and Algorithms.

## How to Read the Documentation

To view the detailed documentation, navigate to the `docs/` directory in this repository.

### For GitHub Visitors:

1.  **Browse Directly:** On GitHub, simply click on the `docs/` folder in the repository's file browser.
2.  **Select a Topic:** Inside the `docs/` folder, you will find markdown files (`.md`) for different topics (e.g., `big-o-notation.md`). Click on any file to view its content directly in your browser. GitHub automatically renders markdown files for easy reading.

For example, to read about Big O Notation, click on `docs/big-o-notation.md`.

We hope you find this documentation helpful!
